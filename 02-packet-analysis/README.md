# LAB 02 - Analisis de paquetes con Wireshark

Capturar y analizar tráfico de red en un entorno controlado para identificar protocolos, patrones de comunicación y posibles riesgos de seguridad.

## Objetivos de este Laboratorio

Capturar y analizar:

- ICMP
- DNS
- TCP Three-Way Handshake
- HTTP

### Herrramientas
- Wireshark
- Kali Linux
- Metasploitable2
