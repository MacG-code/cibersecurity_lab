# cibersecurity_lab

Laboratorio de ciberseguridad personal enfocado en el aprendizaje práctico, análisis de redes, evaluación de vulnerabilidades y prácticas de seguridad defensiva.

## Labs

### 01 - Reconocimiento de Red

- Descubrimiento de host
- Escaneo de puertos
- Deteccion de servicios
- Deteccion de sistema operativo

## Herramientas

- Kali Linux
- Nmap
- Wireshark
- Metasploitable2

## Objetivos

- Desarrollar habilidades prácticas de ciberseguridad
- Documentar los hallazgos de manera profesional
- Construir un portafolio para roles relacionados con la ciberseguridad
