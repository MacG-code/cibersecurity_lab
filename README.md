# Cybersecurity Labs

Este repositorio contiene laboratorios prácticos de ciberseguridad desarrollados para el aprendizaje y la práctica de técnicas de hacking ético y defensa. Cada laboratorio se enfoca en una área específica de la ciberseguridad, desde el reconocimiento de redes hasta la explotación de vulnerabilidades.

## Labs

### 01 - Reconocimiento de Red

- Descubrimiento de host
- Escaneo de puertos
- Deteccion de servicios
- Deteccion de sistema operativo

## Herramientas

- Kali Linux
- Nmap
- Wireshark
- Metasploitable2

## Objetivos

- Desarrollar habilidades prácticas de ciberseguridad
- Documentar los hallazgos de manera profesional
- Construir un portafolio para roles relacionados con la ciberseguridad
